VisStory Maker: Uma Ferramenta de Apoio para Construção de Histórias com Dados

VisStory Maker é uma ferramenta de visualização que oferece um conjunto de módulos para ajudar à construção de histórias com dados. Essas histórias com dados podem ser construídas através de recomendações de visualizações, adição de anotações de fatos de dados e/ou adição de descrição textual de estatísticas descritivas ou percepções interessantes sobre os dados.

Você pode usar o VisStory Maker para construir histórias sobre dados e exportá-las para uma apresentação ou relatório, por exemplo. Os sistemas de recomendações de visualizações e anotações do VisStory Maker utilizam os resultados de estudos de percepção gráfica.
