StoryMaker: Uma Ferramenta de Apoio na Construção de História com Dados

StoryMaker é uma ferramenta de apoio para construção de história com dados. Você pode usar o mecanismo de recomendação de visualização do StoryMaker para encontrar visualizações adequadas para os dados. O sistema de recomendação de visualizações utiliza resultados de experimentos sobre percepção gráfica.
